# Lan-aEnsaio
LançaEnsaio – App mobile (Expo/React Native) para registro rápido e padronizado de ensaios e escalas musicais. Com login seguro, seleção guiada de cidades/instrumentos, gravação com auditoria automática, alerta de correção manual e comprovante do último registro. Backend via Supabase Edge Functions integrado ao Google Sheets.
